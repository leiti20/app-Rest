Documentation API SEPA26

Liste de toutes les opérations disponibles sur le service REST

URL Méthode Description Retour
/ GET Page d'accueil du service HTML
/help GET Documentation de l'API HTML
/sepa26/resume/xml GET Liste des 10 dernières transactions XML
/sepa/resume/html GET Liste des 10 dernières transactions HTML
/sepa26/xml/{id} GET Détail d'une transaction par ID XML
/sepa26/html/{id} GET Détail d'une transaction par ID HTML
/sepa26/insert POST Insertion d'une transaction SEPA validée XSD XML
/sepa26/delete/{id} DELETE Suppression d'une transaction par ID XML
/sepa26/search?date=...&sum=... GET Recherche par date et/ou montant XML
